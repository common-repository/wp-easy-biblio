WP Easy Biblio

== Description ==

WP Easy Biblio es un plugin para Wordpress que permite organizar las citas de las entradas en tu blog al estilo de Wikipedia, usando una version reducida y simple del mismo formato.

Este plugin permite que la informacion de las citas este a pie de pagina, facilitando la lectura del texto y la organizacion de las referencias.

La idea surge basicamente como parte de mi trabajo de final de mis estudios de E.T. en Informatica de Gestio, dentro del marco de Technology Enhanced Learning.

Dentro del trabajo, entre otras actividades, tambien encontramos este otro plugin: WordPress Submit Helper Plugin

La version actual es bastante sencilla y tiene pocas pretensiones, pero cubre la funcionalidad planteada. Hay varias posibilidades e ideas para ampliar este plugin, como seria la comunicacion con sistemas bibliograficos tipo Connotea, o la posibilidad de incluir todas las citas (con filtros) en una pagina.

== Installation ==

1. Upload the 'wp-easy-biblio.php' file to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!
